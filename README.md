# Flask Web App Tutorial

## Setup & Installation

```bash
pip install -r requirements.txt
```

## Running The App

```bash
python main.py
```
## Running The Apitest

```bash
python apiTest.py
```

## Viewing The App
Go to `http://127.0.0.1:5000`
